# Project Background

This project is an English translation of the LangGraph Chinese tutorial. The source tutorial is located in the `archived/dive-into-langgraph/` directory, containing 14 chapters of Jupyter Notebook tutorials.

## About This Project

- **Source Location**: `archived/dive-into-langgraph/`
- **Output Location**: Project root directory `./`

## Standards

- Always use the project root directory as the working directory, do not place project files in `/tmp` or similar locations
- You can create plans in `./docs/plan.md`, remember to write down the core goals to avoid forgetting after multiple conversations
- If there are multiple tasks that need planning, you can create new `./docs/plan-*.md` files
